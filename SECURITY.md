# Security Policy

## Reportando Vulnerabilidades

Se você encontrar uma vulnerabilidade de segurança, **NÃO** abra uma issue pública.

Envie um email para: seguranca@consultasdeveiculos.com

Inclua:
- Descrição da vulnerabilidade
- Passos para reproduzir
- Impacto potencial
- Sugestão de correção (se houver)

Respondemos em até 48 horas úteis.

## Práticas de Segurança

- Tokens de API nunca são logados ou incluídos em mensagens de erro
- Comunicação com a API sempre via HTTPS
- Credenciais são sanitizadas em objetos de erro
- Sem dependências com vulnerabilidades conhecidas
